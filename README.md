# Leetcode-Solver
LeetCode AI Helper 🚀

A Chrome extension that injects an AI-powered assistant directly into LeetCode. It provides an interactive dark-themed popup UI with one-click problem analysis and generates solutions dynamically using Gemini API based on the selected coding language.

✨ Features:
• Floating AI button on LeetCode pages
• Modern dark blue + cyan UI
• Auto-detects problem title and description
• Detects selected coding language automatically
• Generates AI-powered solutions using Gemini API
• Displays approach, code, and complexity analysis
• Error handling and smooth UX

🛠 Tech Stack:
JavaScript, Chrome Extension (Manifest V3), Gemini API, HTML, CSS

Built using VS Code
